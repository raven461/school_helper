Техническая документация проекта "SchoolBot"
Разработка в рамках хакатона «КИБЕР РЫВОК» 
1. Общие сведения

Программное решение представляет собой интеллектуальный сервис (Telegram-бот), предназначенный для оптимизации учебного процесса учащихся школы №419. Система выполняет роль персонального ассистента, агрегируя данные о расписании, заменах и домашних заданиях
2. Функциональные возможности

Реализованный функционал полностью соответствует требованиям технического задания:

    Автоматизированный сбор данных: Модуль парсинга в реальном времени получает актуальную сетку уроков с официального веб-ресурса учебного заведения.

    Обработка динамических изменений: Система отслеживает новостную ленту школы и автоматически вносит корректировки (замены кабинетов/предметов) в отображаемое расписание.

    Интеллектуальная поддержка (AI): Интеграция с языковой моделью Mistral-14B позволяет оказывать тьюторскую поддержку. Бот настроен на объяснение методологии решения, а не на выдачу готового ответа.

    Система управления задачами: Возможность ведения списка домашних заданий с контролем дедлайнов и фиксацией статуса выполнения.

    Геймификация: Внедрена система поощрения в виде начисления опыта (EXP) и выдачи достижений за соблюдение сроков выполнения задач.

3. Технологический стек

    Язык разработки: Python 3.10+

    Framework: aiogram 3.x (асинхронное взаимодействие с API).

    Data Processing: pandas, lxml (структурирование данных из внешних источников).

    Database: SQLite3 (реляционная СУБД для хранения профилей пользователей и прогресса).

    AI Engine: Mistral-14B через API-интерфейс OpenRouter.
	
4. Инструкция по развертыванию
    Подготовка окружения: Установите необходимые зависимости из файла конфигурации:
    pip install -r requements.txt

    Конфигурация среды: Создайте файл .env в корневом каталоге проекта и укажите учетные данные:
	
        BOT_TOKEN — идентификатор Telegram-бота.

        AI_TOKEN — API-ключ доступа к нейросети.

        dev_key — ключ доступа администратора, используется для доступа к командам админиcтратора.

    Инициализация: Запуск сервера бота осуществляется командой:
    python bot.py

5. Руководство пользователя
    Бот находится по ссылке @school_helper_419bot

    Инициация: Команда /start запускает процесс ознакомления с системой.

    Авторизация: Команда /reg необходима для привязки пользователя к конкретному классу и корректной выборки данных из расписания.

    Взаимодействие: Навигация осуществляется через встроенное меню и инлайн-клавиатуры. Для получения консультации AI используется команда /ask.

    Вся справка по командам доступна по команде /help
